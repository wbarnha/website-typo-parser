The image above represents a sequence of orbits taken to steer the initial point (x1,y1) to the origin (0,0) in finite time, when the dynamics are governed by the second order ODE x'' + x = u(t) and u(t) is a control function that that is limited to two values: u = 1 and u = 2. Start from (x1,y1) with u = 2. This steers the initial point along the orbit C1 in the indicated direction. Continue until point P when the value of the control is switched to u = 1. This steers along the orbit O1 again in the indicated direction. When point Q is reached the value of the control is switched back to u = 2. This leads along the orbit C to the origin.
